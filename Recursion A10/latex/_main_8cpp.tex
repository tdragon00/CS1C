This function reverses elements in a char array using two ints and a char array. 

It reverses all elements in the array between the element array\mbox{[}first int\mbox{]} and element\mbox{[}secondint\mbox{]} (inclusive) The function is recursive and it will only reverse letters if the first int is not the second int and the second int is not bigger than the first int.
